Quick Start
# Clone and setup git clone https://github.com/yourteam/smartchain-ai cd smartchain-ai cp .env.example .env.local # Install dependencies (monorepo) pnpm install # Start all services with Docker docker-compose up -d # Seed demo data pnpm db:seed # Run frontend dev server pnpm dev # → http://localhost:3000 # Run AI service (separate terminal) cd apps/ai-service pip install -r requirements.txt uvicorn main:app --reload # → http://localhost:8000

  • REDIS_URL=redis://localhost:6379
